# snake
A Snake clone in C++ that uses `ncurses` library.

# How to compile
* On Ubuntu-based systems

    1. Install `ncurses` library.
        ```
        $ sudo apt install libncurses-dev
        ```

    2. Compile the program.
        ```
        $ make
        ```

    3. Run the program.
        ```
        $ ./snake
        ```
